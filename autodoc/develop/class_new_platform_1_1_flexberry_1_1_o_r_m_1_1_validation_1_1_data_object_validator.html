Проверка, удовлетворяет ли объект данных заданному ограничению.

Аргументы
objectToCheckОбъект данных, для которого применяется ограничение.
limitFunctionФункция ограничения, на удовлетворение которой проверяется объект данных.
Возвращает
true, если объект удовлетворяет переданному ограничению. Иначе false.
Исключения
ArgumentNullExceptionИсключение генерируется при передаче null в качестве значения для objectToCheck .
ArgumentNullExceptionИсключение генерируется при передаче null в качестве значения для limitFunction .
